- This puzzle solver can be ran for the 8-puzzle, 16-puzzle, or 24-puzzle game.
- Initial board is identified by the user. Separate numbers with a single space with the entire board in quotations.
- Implemented in Python3. User will be prompted to enter number choice for BFS, DFS, h1 (misplaced tiles), or h2 (total distance to goal).
- Time taken and moves made will be given at the end of the program. Intermediate steps are shown for all possible paths and associated costs.

- Run: python3 tilesSearch.py "[initialBoard with spaces between values]"
	e.g. python3 tilesSearch.py "1 2 3 4 5 6 7 0 8"
